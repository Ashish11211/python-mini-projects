# 1️⃣ Student Data Program

## 📖 Description
This program takes student details as input and displays them.

## 💻 Code

```python
name = input("enter student name:- ")
roll = int(input("enter roll number: "))
mark = float(input("enter marks: "))

print("\nstudent details")
print("name", name)
print("roll no:", roll)
print("mark", mark)

```
